Attributes

Configuration options for dialog Web Components.

foot-note

  • def - Footnote text (required)
  • title - Modal title (default: "Note")

popup-dialog

  • title - Modal title
  • body - Body text
  • confirm-label - Confirm button text
  • cancel-label - Cancel button text
  • confirm-href - URL on confirm
  • confirm-variant - "accent" or "red"
  • no-cancel - Hide cancel button
