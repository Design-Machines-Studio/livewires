Tables

Data tables with multiple style variants. Use data-th attributes on cells for responsive labeling.

Default table

Clean table styling with minimal borders. Use .numeric for right-aligned number columns.

Code

<table>
  <caption>Table caption</caption>
  <thead>
    <tr>
      <th>Name</th>
      <th class="numeric">Value</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td data-th="Name">Item</td>
      <td data-th="Value" class="numeric">100</td>
    </tr>
  </tbody>
</table>

Selectable rows

Use .checkbox--icon for compact row selection. On narrow screens, checkboxes cluster below the row label.

Code

<table class="table--lined">
  <thead>
    <tr>
      <th>Task</th>
      <th>Review</th>
      <th>Approve</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td data-th="Task">Budget proposal</td>
      <td><label class="checkbox checkbox--icon">
        <input type="checkbox" name="review" value="budget">
        Review budget</label></td>
      <td><label class="checkbox checkbox--icon">
        <input type="checkbox" name="approve" value="budget">
        Approve budget</label></td>
    </tr>
  </tbody>
</table>
