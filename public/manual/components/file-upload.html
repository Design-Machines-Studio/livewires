File upload

Custom file upload inputs with full control over button text, colours, and layout. Includes a drop zone variant for drag-and-drop.

Default

Wraps a hidden file input inside a label. The button text is whatever you put in .file-upload-button.

Code

<label class="file-upload">
  <input type="file" name="resume" accept=".pdf,.doc">
  <span class="file-upload-button">Upload resume</span>
  <span class="file-upload-text">No file selected</span>
</label>

Accent variant

Accent-coloured button for primary upload actions.

Code

<label class="file-upload file-upload--accent">
  <input type="file" name="doc">
  <span class="file-upload-button">Upload document</span>
  <span class="file-upload-text">No file selected</span>
</label>

Compact

Small inline button with no filename area. Uses button--small sizing. The status text is visually hidden but remains accessible to screen readers.

Code

<label class="file-upload file-upload--compact">
  <input type="file" name="import" accept=".csv">
  <span class="file-upload-button">Import CSV</span>
  <span class="file-upload-text">No file selected</span>
</label>

Disabled

Code

<label class="file-upload">
  <input type="file" name="locked" disabled>
  <span class="file-upload-button">Upload</span>
  <span class="file-upload-text">Uploads disabled</span>
</label>

Large

Larger button for prominent upload areas.

Code

<label class="file-upload file-upload--large">
  <input type="file" name="hero">
  <span class="file-upload-button">Choose hero image</span>
  <span class="file-upload-text">No file selected</span>
</label>

Drop zone

Dashed border area for drag-and-drop uploads. Add multiple to accept multiple files.

Code

<label class="file-upload file-upload--zone">
  <input type="file" name="docs" multiple>
  <span class="file-upload-button">Browse files</span>
  <span class="file-upload-text">or drag and drop files here</span>
</label>

Drop zone (accent)

Combine variants for an accent-coloured drop zone.

Native file input

Bare <input type="file"> styled automatically by the base layer. No classes needed.

Code

<input type="file" name="attachment">

Custom properties

Property Default Purpose
--file-upload-button-bg var(--color-fg) Button background
--file-upload-button-fg var(--color-bg) Button text colour
--file-upload-button-h-space var(--button-h-space) Button horizontal padding
--file-upload-button-v-space var(--button-v-space) Button vertical padding
--file-upload-button-font-size var(--button-font-size) Button font size
--file-upload-border var(--color-border) Outer border colour

JavaScript enhancement

The component works without JavaScript. Optionally add this snippet to show the selected filename and enable drag-and-drop feedback.

Code

<script>
// Show selected filename
document.querySelectorAll('.file-upload input[type="file"]')
  .forEach(input => {
    input.addEventListener('change', () => {
      const label = input.closest('.file-upload');
      const text = label.querySelector('.file-upload-text');
      if (input.files.length > 0) {
        const names = Array.from(input.files)
          .map(f => f.name).join(', ');
        text.textContent = names;
        label.setAttribute('data-has-file', '');
      } else {
        text.textContent = 'No file selected';
        label.removeAttribute('data-has-file');
      }
    });
  });

// Drag-and-drop feedback for zone variant
document.querySelectorAll('.file-upload--zone')
  .forEach(zone => {
    ['dragenter', 'dragover'].forEach(evt => {
      zone.addEventListener(evt, e => {
        e.preventDefault();
        zone.setAttribute('data-drag-over', '');
      });
    });
    ['dragleave', 'drop'].forEach(evt => {
      zone.addEventListener(evt, e => {
        e.preventDefault();
        zone.removeAttribute('data-drag-over');
      });
    });
    zone.addEventListener('drop', e => {
      const input = zone.querySelector('input[type="file"]');
      if (input && e.dataTransfer.files.length) {
        const dt = new DataTransfer();
        Array.from(e.dataTransfer.files)
          .forEach(f => dt.items.add(f));
        input.files = dt.files;
        input.dispatchEvent(
          new Event('change', { bubbles: true })
        );
      }
    });
  });
</script>
