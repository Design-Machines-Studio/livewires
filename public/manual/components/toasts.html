Toasts

Dismissible notification messages for user feedback.

Default toast

A neutral notification with a close button.

Code

<div class="toast">
  <p class="toast__content">Something happened.</p>
  <button class="toast__close" aria-label="Dismiss">&times;</button>
</div>

Toast container

Stack toasts in a fixed-position container at the top right of the viewport.

Code

<div class="toast-container">
  <div class="toast toast--success">
    <p class="toast__content">Toast 1</p>
    <button class="toast__close" aria-label="Dismiss">&times;</button>
  </div>
  <div class="toast toast--info">
    <p class="toast__content">Toast 2</p>
    <button class="toast__close" aria-label="Dismiss">&times;</button>
  </div>
</div>

The container uses position: fixed to anchor toasts to the top-right corner.
