Dropdown

Toggleable overlay menus for actions and navigation.

Basic dropdown

Toggle via .is-open class or [open] attribute on the container.

Code

<div class="dropdown is-open">
  <button class="button dropdown__trigger">Menu</button>
  <ul class="dropdown__menu">
    <li><a href="#">Action 1</a></li>
    <li><a href="#">Action 2</a></li>
    <li><hr></li>
    <li><button>Delete</button></li>
  </ul>
</div>

Menu dividers

Use <hr> inside list items to separate groups of actions.

Code

<ul class="dropdown__menu">
  <li><a href="#">View</a></li>
  <li><a href="#">Edit</a></li>
  <li><hr></li>
  <li><a href="#" disabled>Unavailable</a></li>
</ul>
  • View
  • Edit
  • Unavailable

Alignment

End-align the menu with .dropdown--end. Open upward with .dropdown--up.

Code

<div class="dropdown dropdown--end">...</div>
<div class="dropdown dropdown--up">...</div>
