Color schemes

Pre-defined color combinations for backgrounds, text, and accents. Apply to any container to change its color context.

Available schemes

Each scheme sets 8 semantic tokens: --color-bg, --color-fg, --color-accent, --color-accent2, --color-muted, --color-subtle, --color-border, --color-border-muted.

Code

<!-- Mode schemes (dark-mode adaptive) -->
<section class="scheme-default">...</section>
<div class="scheme-subtle">...</div>

<!-- Colour schemes -->
<div class="scheme-purple">...</div>
<div class="scheme-black">...</div>
<div class="scheme-red">...</div>
<div class="scheme-orange">...</div>
<div class="scheme-gold">...</div>
<div class="scheme-green">...</div>
<div class="scheme-blue">...</div>

<!-- Status schemes -->
<div class="scheme-danger">...</div>
<div class="scheme-warning">...</div>
<div class="scheme-success">...</div>
