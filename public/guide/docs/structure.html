Project structure

Each Live Wires installation is a single prototype. No templates, no build complexity. Just HTML, CSS, and a simple Web Component for includes.

Directory overview:

/src/css/
All CSS source files organized by cascade layers (tokens, elements, layouts, utilities)
/src/css/1_tokens/
Design tokens: spacing.css, typography.css, color.css
/src/css/4_elements/
Semantic HTML defaults (headings, paragraphs, links, lists, forms, tables)
/src/css/5_layouts/
Layout primitives: box, stack, grid, cluster, sidebar, center, and section
/src/css/7_utilities/
Utility classes: spacing, typography, color
/src/main.js
Vite entry point (loads CSS and Web Components)
/src/html-include.js
Zero-dependency Web Component for HTML includes
/public/
Your prototype lives here. Build HTML pages directly in this folder.
/public/_includes/
Reusable HTML fragments (header, footer, etc.) used with <html-include>. Customize and covert easily for your own front-end system.
/public/fonts/
Web font files
/public/img/
Images and graphics
/public/guide/
This style guide (you're reading it). Also functions as a component library.
