Test Layout Page

This demonstrates the new html-include layout system.

Introduction

This page uses the guide layout with the new include pattern. The layout is split into two files:

  • /_includes/layouts/guide-before-content.html
  • /_includes/layouts/guide-after-content.html

Features

This approach gives you:

  • Minimal page files (just content between includes)
  • DRY layouts (update once, applies everywhere)
  • No build step required
  • SEO-friendly (content is in HTML)
