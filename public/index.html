Live Wires 2026

A prototyping-first design system for editorial websites. Built with modern CSS, rooted in timeless design principles.

Philosophy: Good Defaults + Additive Art Direction

This page demonstrates that semantic HTML looks good with minimal classes. The baseline typography, spacing, and layout just work.

Live Wires is not a component library. It's a design philosophy and workflow tool that helps you:

  • Prototype quickly in HTML & CSS
  • Systematize patterns as you discover them
  • Add art direction through utility classes
  • Maintain perfect vertical rhythm

The Sacred Baseline

Everything in Live Wires derives from the --line variable. Change the base font size or line height, and the entire system recalculates. This ensures perfect vertical rhythm across your design.

Good design is as little design as possible. Start with sensible defaults, then add classes for art direction.
