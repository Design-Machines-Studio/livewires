Live Wires

Prototyping-first design for editorial websites

Built with modern CSS. Rooted in a philosophy where nothing gets thrown away—the prototype becomes the product.

The problem with design handoffs

Web design has traditionally been wasteful. We spend hours in Figma crafting comps that get handed off and translated. We configure Tailwind only to fight against it later. We build wireframes in tools that don't reflect reality, then throw them away when the next phase begins.

How much better would our work be if we could turn those hours into real design thinking and execution?

“Instead of creating a series of disposable deliverables, the design process becomes an evolution.”

The solution: prototype-first

Live Wires takes a different approach: everything you build takes you closer to the final product. Start with real content in real HTML. The baseline styles make it look good immediately. Adjust tokens to establish your foundation. Layer utilities for art direction. Extract components as patterns emerge.

Nothing is throwaway. Each step builds on the last.

The sacred baseline

Everything in Live Wires derives from a single variable: --line. Change the base font size or line height, and the entire system recalculates—spacing, typography, layout. This ensures perfect vertical rhythm across your design.

Layout primitives

Instead of traditional grid systems, Live Wires uses compositional layout primitives inspired by Every Layout. Combine them to create complex layouts without writing custom CSS.

Stack

Vertical spacing between elements. The fundamental building block.

Grid

Auto-responsive grid layouts. No breakpoints needed.

Cluster

Horizontal groups with wrapping. Perfect for tags and nav.

Center

Horizontally centered content with max-width constraints.

Sidebar

Main content with sidebar. Collapses on narrow screens.

Section

Full-width with responsive padding. Container query ready.
