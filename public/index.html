Live Wires

Prototyping-first design for editorial websites

The progressive design system that takes you from prototype to launch. Built with modern CSS. No pre-processors. No frameworks. No fuss. Nothing gets thrown away—the prototype becomes the product.

The problem with design handoffs

Web design has traditionally been wasteful. We spend hours in Figma crafting comps that get handed off, translated to code, then discarded like yesterday's trash. We go hard on utility classes or over-designed frameworks only to fight against it later. We build wireframes in tools that don't reflect reality, then throw them away when the next phase begins.

How much better would our work be if we could turn those hours into real design thinking and execution?

“Instead of creating a series of disposable deliverables, the design process becomes an evolution.”

The process: carving a sculpture

Every step builds on the last. Start with raw material. Shape the form. Refine the details. Add expression.

1

Prototype in HTML

Build page templates using semantic HTML and layout primitives. Your prototype is instantly viewable in real browsers across devices. Share with clients. Test UX flows. Redact the text to focus on interaction without the distraction of words.

Nothing replicates the browsing experience like actual web pages.

2

Set your tokens

Adjust the design tokens—colors, typography, spacing—and watch the Manual transform in real time. Every token you set is designing the foundation of your project.

The system adapts. You see the results immediately.

3

Refine and express

Expand your CSS. Add utility classes for art direction. Refine components. Each iteration takes you from coarse to detailed—like carving marble.

A stable foundation with room for expression.

4

Apply to production

Connect your templates to your CMS or framework of choice. Your prototype becomes the product. Nothing gets thrown away.

The same code, evolved—not translated.

Alternative: CMS-first

For an even more agile approach, install Live Wires directly on your CMS and start content modeling immediately with a prototype front-end. Then continue with step 2. CMS starters coming soon.

The sacred baseline

Everything in Live Wires derives from a single variable: --line. Change the base font size or line height, and the entire system recalculates—spacing, typography, layout. This ensures perfect vertical rhythm across your design.

Layout primitives

Instead of traditional grid systems, Live Wires uses compositional layout primitives inspired by Every Layout. Combine them to create complex layouts without writing custom CSS.

Stack

Vertical spacing between elements. The fundamental building block.

Grid

Auto-responsive grid layouts. No breakpoints needed.

Cluster

Horizontal groups with wrapping. Perfect for tags and nav.

Center

Horizontally centered content with max-width constraints.

Sidebar

Main content with sidebar. Collapses on narrow screens.

Section

Full-width with responsive padding. Container query ready.

Utility classes

Tailwind-compatible utilities for precise control. All utilities use the utilities cascade layer, so they always win over component and layout styles.
